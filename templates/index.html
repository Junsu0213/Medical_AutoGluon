Select a Data Type

Tabular (Classification)

This tool helps you classify tabular data using AutoGluon.

Input Data:

  • CSV file with features and target column
  • Target column should contain categorical values

Output:

  • Trained classification model
  • Performance metrics
  • Feature importance analysis

Tabular (Regression)

This tool helps you predict continuous values using AutoGluon.

Input Data:

  • CSV file with features and target column
  • Target column should contain numerical values

Output:

  • Trained regression model
  • Performance metrics (RMSE, MAE, R²)
  • Feature importance analysis

Image (Classification)

This tool helps you classify images using AutoGluon.

Input Data:

  • Folder of images organized by class
  • Supported formats: JPG, PNG

Output:

  • Trained image classification model
  • Classification accuracy and other metrics
  • Class prediction probabilities

Image (Object Detection)

This tool helps you detect and locate objects in images using AutoGluon.

Input Data:

  • Images with bounding box annotations
  • COCO or VOC format annotations

Output:

  • Trained object detection model
  • Bounding box predictions
  • Detection confidence scores
